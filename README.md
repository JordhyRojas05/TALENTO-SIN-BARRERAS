# TALENTO SIN BARRERAS 🚀

## 📌 Descripción

**Talento Sin Barreras** es una aplicación web desarrollada con tecnologías modernas como React, TypeScript y Tailwind CSS, cuyo objetivo es proporcionar una plataforma digital accesible para la gestión de usuarios y campañas.

El sistema permite a los usuarios registrarse, iniciar sesión y acceder a un panel principal (dashboard) desde donde pueden crear, visualizar y administrar campañas, así como gestionar su perfil personal. La aplicación está diseñada con un enfoque en la usabilidad y la inclusión, buscando facilitar la interacción de diferentes tipos de usuarios dentro de un entorno intuitivo y funcional.

Este proyecto representa un prototipo frontend que puede ser escalado e integrado con servicios backend para ofrecer funcionalidades completas en un entorno real.

---

## 🎯 Objetivo General

Desarrollar una plataforma web que facilite la gestión de usuarios y campañas, promoviendo la accesibilidad e inclusión digital.

---

## 🎯 Objetivos Específicos

- Permitir el registro e inicio de sesión de usuarios.
- Implementar un dashboard interactivo.
- Facilitar la creación y edición de campañas.
- Gestionar perfiles de usuario.
- Ofrecer una interfaz amigable e intuitiva.

---

## 🛠️ Tecnologías utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- Radix UI
- React Hook Form
- Recharts
